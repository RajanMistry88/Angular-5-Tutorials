Forgot Password Component

Forgot Password page we verify the User Mobile No if it is Valid then we send OTP SMS to his/her Number after verify the OTP Application. allow user to Gengerate New Password. In This Applicaiton We have not intigrated any SMS Service so we are not verify the OTP. If Mobile No is Valid then We Allow user to change His/Her Password. And In Real time Application you have to integrate SMS or Email Services For Security.

    ForgotPassword html component we use form tag which is use for validation.

  • 2. ForgotPassword.component.ts('Find Component inside src/pages/ForgotPassword/ Folder » ForgotPassword.component.ts')

    we create 2 method in this component 1. ngOnInit() for Validation and 2. fnforgotpassword() for verification if user Mobile no is register or not. In this component we share all the input data which are coming from View to the WebAPI and use POST Request.
