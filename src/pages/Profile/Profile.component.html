Profile Component

ProfileInfo page is allow user to update his/her recode. We store UserID in Session when user login to the application so using that ID we verify user and get all information and display in the Form.

  • 1. Profile.component.html('Find Component inside src/pages/Profile/ Folder » Profile.component.html')

    Profile html component we use form tag which is use for validation.

  • 2. Profile.component.ts('Find Component inside src/pages/Profile/ Folder » Profile.component.ts')

    we create 3 method in this component 1. ngOnInit() for Validation and 2. fuGetProfile() will get all the user information and display on the form 3. fnProfileUpdate() will Edit and Update the user information. In this component we share all the input data which are coming from View to the WebAPI and use PUT Request to update recode. to get detail of the user we use GET Request

  • 3. Profile.component.css ('Find Component inside src/pages/Profile/ Folder » Profile.component.css')

    if you want to create custom Css for this page you can create over here, so it will only affect on this page
