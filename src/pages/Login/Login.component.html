Login Component

Login page is use for Authenticate the user. So in this page we created 3 component which are lsited below.In this Application we are using WebAPI for Authentication.

  • 1. Login.component.html('Find Component inside src/pages/Login/ Folder » Login.component.html')

    Login html component we use form tag which is use for validation. you can see in this page.

  • 2. Login.component.ts('Find Component inside src/pages/Login/ Folder » Login.component.ts')

    2 method in this component 1. ngOnInit() for Validation and 2. fnlogin() for verification In this component we share all the input data which are coming from View to the WebAPI and use POST Request.
    If user is valid then API Response the Status in Json Format and user will redirect to another Page Where he/she can access the other Functionality of the application like Profile update, Change Password, Delete Account,Deactivate Account etc.

  • 3. Login.component.css ('Find Component inside src/pages/Login/ Folder » Login.component.css')

    if you want to create custom Css for this page you can create over here, so it will only affect on this page
