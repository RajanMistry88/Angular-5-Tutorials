Registration Component

Registration page is the basic need of any Application. we take user information with that they access application features. So in this page we created 3 component which are lsited below.

  • 1. Registration.component.html('Find Component inside src/pages/Registration/ Folder » Registration.component.html')

    Registration html component we use form tag which is use for validation. you can see in this page.

  • 2. Registration.component.ts('Find Component inside src/pages/Registration/ Folder » Registration.component.ts')

    2 method in this component 1. ngOnInit() for Validation and 2. fnregistration() to insert user recode in Database. In this component we share all the registration Form input data of to the WebAPI and use POST Request.

  • 3. Registration.component.css ('Find Component inside src/pages/Registration/ Folder » Registration.component.css')

    if you want to create custom Css for this page you can create over here, so it will only affect on this page
