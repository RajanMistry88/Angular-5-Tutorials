Application Information

Create Application Environment

1st thing you have to Create DataBase to start this Tutorial Download the DataBase file and Restore in your Database or download Script file and Run in Your Sql Server so it will Automatic Creatre everything.

2nd Download or Clone WebAPI Open Project and generate or change connection string in your webconfig file, there are 2 ways 1. write down in your Web.config File, 2. Use ADO.net Entity-Framework. Once it's done Run API in local and copy the API path ('http://localhost:62952/api').
Note: If you want to use this API for other project, then you have to host this webAPI in hosting services and use that domain name('http://www.mvcwebapi.com/api') to the Angular Project.

3rd thing you have to do is Download or Clone Angular-5 (or Zip file) Project. Open Folder src/provider/ » HttpManager.module.ts file. the WebAPI path you have copied replace with this path ('URL: string = "http://localhost:62952/api"').
Note: Make sure when you publish or your local path is change you just need to change only this path for example if you publish this API on server then enter domain name : URL:string = "http://www.mvcwebapi.com/api" in HttpManager.module.ts file.

Application Coding Information

In this Application for all the module we Created 3 Component 1. CssComponent filename.component.css, 2. HtmlComponent filename.component.html, 3. TsComponent filename.component.ts

    Component

    Components are basically classes that interact with the .html file of the component, which gets displayed on the browser.

  • 1. app.component.css

    here you can define the Global Css for all the component in the project.

  • 2. app.component.html

    in this component we create Global html design. For Designing Information Click here

  • 3. app.component.ts

    over here we can write our applicaiton algorithm.

  • 3. app.module.ts

    in this file we import all the component and provider which we want to use in project.

    • Services

    In this Project we use Services which is in the provider Folder. we use Dependency-injection for this services and it is use for entier Project. For more info click here

  • 1. GlobalManager.module.ts

    In this Service we create 2 mehtod.
    1. fnCheckSession() this funcation check the user session and redirect as per that.
    2. fnClearSession() this funcation Clear all Session.

  • 2. HttpManager.module.ts

    In this Service we are using REST full WebAPI. to use this webapi in our project we import HttpClient using that we Create GET, POST, PUT, DELETE method. For more info click here

  • 3. RoutingManager.module.ts

    In this Service we import RouterModule to route our component. also import the components. For more info click here

  • 4. ValidationManager.module.ts

    In this Service we import forms for validation then we create all validation message in validationMessage() funcation.
    To Check mobile no, email format etc. we use regular expression. For more info click here
