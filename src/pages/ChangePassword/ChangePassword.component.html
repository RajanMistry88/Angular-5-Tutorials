Change Password Component

Change Password page is allow user to create new Password. So in this page we created 3 component which are lsited below.

  • 1. ChangePassword.component.html('Find Component inside src/pages/ChangePassword/ Folder » ChangePassword.component.html')

    ChangePassword html component we use form tag which is use for validation. you can see in this page.

  • 2. ChangePassword.component.ts('Find Component inside src/pages/ChangePassword/ Folder » ChangePassword.component.ts')

    we create 2 method in this component 1. ngOnInit() for Validation and 2. fnchangepassword() will allow user to change here old password. In this component we share all the input data which are coming from View to the WebAPI and use POST Request.

  • 3. ChangePassword.component.css ('Find Component inside src/pages/ChangePassword/ Folder » ChangePassword.component.css')

    if you want to create custom Css for this page you can create over here so it will only affect on this page
