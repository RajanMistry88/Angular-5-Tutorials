In this Project we are using Simple Bootstrap design. visit this Design.component.html (Folder Src » pages » Design » Design.component.html) page for better understanding how to apply css.
To Use this Default bootstrap Css in other Projects Follow this step
1. npm install bootstrap@3.3.7 hit this command install the package.
2. Open .angular-cli.json file and press Ctrl+F Search "styles" and "scripts" find bootstrap file location and add css into style "../node_modules/bootstrap/dist/css/bootstrap.css"
3. Or add CDN link in index.html file
